# PWA Quiz

## Getting Started

bash```yarn install```

bash```yarn dev```

## Info

PWA Quiz app

You can find data to quiz in `quiz-data` folder

Quiz format:

```json
{
   "data": [
      {
         "id": string,
         "title": string,
         "options": Array<string>,
         "answer": string
      }
   ]
}
```

It must be stringified.
